Прецедент создание проекта

Основной поток:
1. Прецедент начинается когда company root выберает опцию "Создать проект".
2. Система предлагает пользователю company root ввести название проекта и описание.
3. Система предлагает пользователю company root выбрать project root по email.
4. Система предлагает пользователю company root подключить внешние системы к проекту
5. Система предлагает пользователю company root добавить в проект команды из существующих в компании.

Альтернативные потоки:
1. Альтернативный поток начинается после шага 5 основного потока.
 1.1. Если необходимой команды нет в списке команд компаннии, система предлагает пользователю 
      company root создать новую команду

	  
	  
Прецедент перетаскивание задачи

Основной поток:
1. Прецедент начинается когда пользователь user хочет перетащить задачу из одной 
   колонки в доске в другую
2. Пользователь user перетаскивает задачу из текущей колонки в целевую
3. Задача  меняет свой статус и отображается в другой колонке

Альтернативные потоки
1. Альтернативный поток начинается после шага 2 основного потока.
 1.1. Если у пользователя нет необходимых прав на перетаскивание задачи, система выводит
      предупреждающее сообщение
2. Альтернативный поток начинается после шага 2 основного потока.
 2.1. Если в настройках доски не указана возможность перетаскивать задачи из текущей колонки
      в целевую, система выводит предупреждающее сообщение
3. Альтернативный поток начинается после шага 2 основного потока.
 3.1. Если текущая задача заблокирована по причине присутствия в текущей колонки связанной задачи,
      требующей выполнения в первую очередь, система выводит предупреждающее сообщение.
